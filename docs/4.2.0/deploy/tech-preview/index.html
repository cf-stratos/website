Tech Preview

Some features that are close to being production ready but require feedback from the community are categorised as being in 'Tech Preview'. These features are distributed in Stratos however are disabled by default.

Enable Tech Preview#

To enable these features the Environment Variable ENABLE_TECH_PREVIEW or helm chart value console.techPreview should be set to true.
